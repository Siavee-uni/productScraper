### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Siavee-uni/productScraper.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Change the example.env to .env`
4. Enter your telegram bot token and friend keys to .env`
```JS
Tele_Token = 'ENTER YOUR Token';
```
